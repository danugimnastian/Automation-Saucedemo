# SauceDemo Automation Testing

This repository contains automation tests for the [SauceDemo](https://www.saucedemo.com/). The tests are designed to verify the functionality of the website's key features, ensuring a smooth and reliable for user experience.

## Features
- Login
- Sorting Items
- Add Items to Cart
- Checkout
- Logout
